プライバシーポリシー

最終更新日: 2026年4月12日

横田裕市(以下「当方」といいます)は、iOSアプリケーション「Yorishiro」(以下「本アプリ」といいます)におけるユーザーの個人情報の取り扱いについて、以下のとおりプライバシーポリシー(以下「本ポリシー」といいます)を定めます。

第1条(基本方針)

当方は、ユーザーのプライバシーを尊重し、個人情報の保護に関する法律(個人情報保護法)その他の関連法令を遵守し、適切に個人情報を取り扱います。

第2条(取得する情報)

本アプリでは、以下の情報を取得します。

1. 匿名ユーザーID

Supabase Anonymous Authenticationによって内部的に発行される識別子です。氏名、メールアドレス、電話番号など個人を特定する情報は含みません。記録データ、課金状態、利用制限管理のためにのみ利用されます。

2. ニックネーム

ユーザーが任意で設定した場合のみ取得します。設定は任意であり、設定しなくても本アプリの主要機能はご利用いただけます。

3. ジャーナル本文・自由記述

ユーザーが入力した日記本文、AIリライト後の文、対話分析の入力、プロフィール生成に使用される文章を保存します。

4. 感情・思考・コンディションに関する自己申告データ

気分、精神面、体力スコア、思考の傾向、セルフケアの文脈など、ユーザーが自己申告した情報を保存します。

5. 購入履歴・プラン状態

月額/年額/買い切りプランの購入状態、Apple StoreKitのトランザクション情報、Entitlement(権利)同期情報を保存します。クレジットカード情報など決済情報そのものは取得しません(決済はApple経由で行われ、当方はそれらにアクセスできません)。

6. 利用状況データ

AI機能の利用回数や利用種別など、機能制限管理および品質向上に必要な範囲のデータを取得します。

7. 標準的なアプリ解析データ

Apple App Analyticsを通じた標準的なアプリ利用統計情報(クラッシュレポート、利用デバイス、OSバージョンなど)を取得する場合があります。これらはAppleによって集計され、当方は個別のユーザーを特定できる形で受け取ることはありません。

第3条(利用目的)

取得した情報は、以下の目的でのみ利用します。

  1. 本アプリの提供および機能の運用
  2. ユーザーが入力したテキストに対するAI分析機能の提供
  3. 利用状況の管理および機能制限の制御
  4. 課金プランの管理および購入履歴の同期
  5. 不正利用の防止
  6. お問い合わせへの対応
  7. 本アプリの改善および新機能開発
  8. 法令または規制当局からの要請に対応する場合の対応

第4条(AI機能とデータの送信)

1. 利用するAIサービス

本アプリのAI機能は、Anthropic, PBC.が提供するClaude API(https://www.anthropic.com)を利用しています。

2. 送信されるデータ

ユーザーが入力したジャーナル本文、対話分析の入力テキスト、プロフィール生成のためのデータが、AI分析処理のためにAnthropicのAPIサーバーに送信されます。

3. データの取り扱い

Anthropic, PBC.は、APIを通じて受信したデータをモデルのトレーニングに使用しないことを公式に表明しています。最新の取り扱いについては、Anthropic, PBC.のプライバシーポリシーをご確認ください。

4. ユーザーの選択

ユーザーは、本アプリのAI機能を利用しない選択も可能です。ただしAI機能を利用しない場合、本アプリの主要機能の多くがご利用いただけなくなる点をご了承ください。

第5条(データの保管と管理)

1. 保管先

本アプリのデータは、Supabaseの東京リージョン(Northeast Asia / Tokyo)のデータベースに保管されます。

2. セキュリティ対策

通信はすべてHTTPSで暗号化されます。APIキーはサーバー側のみで管理され、クライアント(アプリ)からは直接アクセスできない構造になっています。

3. バックアップ

データはSupabaseの標準的なバックアップポリシーに従って自動バックアップされます。バックアップは障害復旧およびデータ保護を目的としており、不必要に長期間保持することはありません。

4. データ保管期間

ユーザーが本アプリを利用している間、データは継続して保管されます。ユーザーが削除を希望された場合、本ポリシー第7条に従って削除されます。

第6条(第三者提供)

当方は、以下の場合を除き、取得した情報を第三者に提供しません。

  1. ユーザーの同意がある場合
  2. 法令に基づく場合
  3. 人の生命、身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難であるとき
  4. 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、本人の同意を得ることが困難であるとき
  5. 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合であって、本人の同意を得ることにより当該事務の遂行に支障を及ぼすおそれがあるとき

なお、AI分析のためにAnthropic, PBC.のAPIへデータを送信することは、サービス提供の本質的な構成要素であり、ユーザーが本アプリを利用することにより同意したものとみなします。

第7条(ユーザーの権利とデータ削除リクエスト)

1. 開示・訂正・利用停止・削除の請求

ユーザーは、ご自身の情報について、開示・訂正・利用停止・削除を請求することができます。

2. データ削除リクエストの対応

ユーザーからのデータ削除のご依頼には、本人確認の手続きを経た上で、原則として30日以内に対応します。当方は可能な限り迅速に対応するよう努めます。

3. 例外

以下の場合、データの一部または全部が削除対象外となることがあります。

  • 課金処理・サブスクリプション確認のために保管が必要な情報
  • 不正利用防止のために必要な情報
  • 法令により保管が義務付けられている情報

4. 対応の遅延

本人確認に時間を要する場合、または対応に技術的な検討が必要な場合など、30日以内に対応できない場合は、お申し出いただいた方にその理由および見込み対応時期をお知らせします。

5. 依頼窓口

データ削除その他の権利行使のご依頼は、以下までご連絡ください。

メールアドレス: main@yokoichi.jp

第8条(アプリ内のデータ管理機能)

本アプリには、ユーザーご自身でデータを管理いただける以下の機能を備えています。

  1. Insights機能のオン/オフ切替 - 蓄積データの分析機能を無効化できます
  2. 人物名・場所名の自動抽出のオン/オフ切替 - AIによる人物・場所の自動抽出を停止できます
  3. 除外リスト - 自動抽出から除外したい人物名・キーワードを登録できます
  4. 期間別データ削除 - 特定期間のデータを個別に削除できます
  5. カテゴリ別データ削除 - 人物・場所などカテゴリごとにデータを削除できます
  6. 全データ削除 - すべてのInsightsデータをまとめて削除できます

第9条(年齢制限と未成年者の情報)

  1. 本アプリは13歳以上の方を対象としています。13歳未満の方は本アプリをご利用いただけません。
  2. 18歳未満の未成年者が本アプリを利用する場合、保護者の同意を得た上でご利用ください。
  3. 未成年者の方は、必要に応じて保護者等の信頼できる大人と相談しながらご利用ください。
  4. 当方が13歳未満の方の情報を誤って取得した可能性があると判明した場合、速やかに該当情報を削除します。

第10条(重要なお知らせ・メンタルヘルスについて)

医療助言ではないことについて

本アプリは、日々の振り返りや自己理解を支援するためのものであり、医療、診断、治療、または精神科的助言を提供するものではありません。気分の落ち込みや不安が強い場合、または緊急の支援が必要な場合は、本アプリの利用のみに依存せず、保護者、医療機関、または適切な相談窓口にご相談ください。未成年の方は、必要に応じて保護者等の信頼できる大人と相談しながらご利用ください。

第11条(クッキー等の技術)

本アプリはモバイルアプリケーションであり、ウェブブラウザのクッキーは使用しません。ただし、アプリの動作に必要な範囲で、デバイス内のローカルストレージ等を使用する場合があります。

第12条(国外への情報移転について)

  1. 本アプリのAI機能は、Anthropic, PBC.のAPIを利用しており、AnthropicはAmerica合衆国に本社を置く事業者です。ユーザーが入力したテキストデータは、AI処理のためにAnthropicのサーバー(アメリカ合衆国を含む地域)に送信される場合があります。
  2. 本アプリのデータベースは、Supabaseの東京リージョンに保管されています。

第13条(プライバシーポリシーの変更)

  1. 本ポリシーの内容は、法令その他本ポリシーに別段の定めのある事項を除いて、ユーザーに通知することなく、変更することができるものとします。
  2. 本ポリシーの重要な変更については、アプリ内通知によりお知らせします。
  3. 当方が別途定める場合を除いて、変更後のプライバシーポリシーは、本アプリ上に掲載したときから効力を生じるものとします。
